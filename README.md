# 🌤 Advanced Weather App

A modern and responsive Weather App built using:

- HTML
- CSS
- JavaScript
- OpenWeatherMap API

This app provides real-time weather information with a beautiful glassmorphism UI, animated background, smooth scrolling, and responsive design.

---

# ✨ Features

✅ Search weather by city  
✅ Current location weather  
✅ Animated gradient background  
✅ Glassmorphism UI  
✅ Responsive design  
✅ Horizontal forecast scroll section  
✅ Weather icons  
✅ Smooth scrolling  
✅ Custom scrollbar  
✅ Enter key support  
✅ Hover animations
